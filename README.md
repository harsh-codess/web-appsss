# Web Development Projects

A collection of web development practice projects focusing on HTML and CSS fundamentals.

## 📁 Project Structure

```
web-development/
├── basics/                       # Foundational CSS concepts
│   ├── box-model/               # CSS Box Model exercises
│   ├── css-basics/              # Introduction to CSS
│   ├── fonts/                   # Typography and font styling
│   └── meme-prod/               # Meme production project
├── 6-motivation-meme/           # Motivation meme creation
├── 7-css-cascade/               # CSS cascade, specificity, and inheritance
└── 7.1-combining-selectors/     # CSS selector combinations
```

## 🎯 Projects Overview

### Basics

#### Box Model
Understanding the CSS box model - margin, border, padding, and content.

#### CSS Basics
Introduction to CSS syntax, selectors, and basic styling properties.

#### Fonts
Exploring typography with `font-family`, `font-size`, and text styling properties.

#### Meme Production
Creating meme layouts using HTML and CSS positioning.

### Advanced Concepts

#### 7. CSS Cascade
Learning about:
- CSS Specificity rules
- Cascade and inheritance
- ID vs Class vs Element selectors

#### 7.1 Combining Selectors
Mastering CSS selector combinations:
- Descendant selectors
- Child selectors
- Multiple selectors
- Class and element combinations

#### 6. Motivation Meme
Creating motivational meme posters with positioning and styling.

## 🚀 Getting Started

1. Clone this repository:
```bash
git clone https://github.com/harsh-codess/devlopment-of-web-applications.git
cd devlopment-of-web-applications
```

2. Open any project folder in your browser:
   - Use Live Server extension in VS Code, or
   - Open the `index.html` file directly in your browser
